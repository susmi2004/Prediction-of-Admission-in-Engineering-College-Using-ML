About Us

Welcome to our College Prediction System! This project uses machine learning algorithms to help students predict their college admission chances based on various factors.

Our system takes into account multiple parameters such as:

  • Preferred College
  • Preferred Branch
  • Gender
  • Category
  • Rank

Using these inputs, our ML model provides accurate predictions to help students make informed decisions about their college applications.
